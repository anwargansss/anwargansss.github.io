# 🚀 Day Challenge: Grow Every Day

**Day Challenge** adalah platform *self-development gamification* berbasis AI yang dirancang untuk membantu pengguna membangun kebiasaan positif melalui tantangan harian yang interaktif, akuntabilitas sosial, dan sistem reward yang memotivasi.

## 🌟 Core Features
* **Gamified Daily Challenges:** Selesaikan misi harian untuk mendapatkan poin dan meningkatkan *level* (Beginner to Legend).
* **AI-Driven Motivation:** Pengingat personal dan *insight* harian untuk menjaga konsistensi pengguna.
* **Verified Anti-Cheat System:** Validasi progres melalui durasi tonton video materi secara real-time.
* **Accountability Partners:** Fitur untuk saling memantau progres dengan partner challenge.
* **Premium Ecosystem:** Akses ke konten eksklusif, statistik lanjutan, dan fitur kustomisasi (Dark Mode, Badge) dengan penukaran poin atau langganan.

## 🛠 Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Modern & Responsive UI)
* **Logic:** JavaScript (DOM Manipulation, State Management)
* **Integration:** YouTube IFrame Player API (Video Validation)
* **Platform:** GitHub Pages (Deployment)

## 📈 Monetization Strategy
* **Freemium Model:** Akses gratis untuk fitur utama.
* **Premium Access:** Rp20.000/bulan | Rp75.000/6 bulan | Rp120.000/tahun.
* **Loyalty Points:** Pengguna dapat membuka akses Premium secara gratis dengan mengumpulkan 500 poin hasil dari konsistensi.

## 🤝 Project Vision
Kami percaya bahwa pengembangan diri haruslah menyenangkan. **Day Challenge** mengubah rutinitas yang membosankan menjadi ekosistem digital yang suportif, modern, dan memberikan dampak nyata bagi setiap penggunanya.
